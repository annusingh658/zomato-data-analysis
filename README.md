# 🍽️ Zomato Data Analysis

## 📊 Project Overview

This project focuses on analyzing Zomato restaurant data to understand customer preferences, restaurant types, and the impact of online ordering on votes and engagement.
The goal is to extract meaningful insights using data analysis and visualization techniques.

---

## 🎯 Objectives

* Analyze different types of restaurants
* Study the impact of online ordering on votes
* Identify popular restaurant categories
* Visualize trends using charts and heatmaps

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📁 Dataset

The dataset contains information about:

* Restaurant types
* Votes
* Online order availability
* Other relevant attributes

---

## 📈 Key Analysis Performed

* Data Cleaning & Preprocessing
* GroupBy Analysis
* Pivot Tables
* Heatmap Visualization
* Bar Charts & Line Graphs

---

## 🔥 Key Insights

* Restaurants with online ordering tend to receive more votes
* Certain restaurant types are more popular among users
* Customer engagement varies significantly across categories

---

## 📌 Conclusion

This project demonstrates how data analysis helps in understanding business trends and customer behavior in the food industry.
